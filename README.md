# ✨ FunFun — 超可愛互動網頁小作品

> 兩個用純 HTML + CSS + JavaScript 打造的絢麗可愛網頁，零依賴、直接開啟即可使用！

---

## 🗂️ 專案內容

### 💖 愛心.html — 互動式愛心動畫

用 Canvas 參數方程式繪製的浪漫愛心，充滿粒子與光效。

**效果特色**
- 🌌 深紫星空背景，150 顆閃爍星星
- 💓 心跳脈動動畫（漸層填色 + 光暈呼吸）
- ✨ 600 顆內部閃爍光點 + 220 顆輪廓粒子沿邊緣運行
- 🌸 背景粒子從底部持續飄升

**互動方式**

| 操作 | 效果 |
|------|------|
| 🖱️ 滑鼠移動 | 粒子被推開，形成漣漪 |
| 🖱️ 按住滑鼠 | 更強排斥力 |
| 👆 點擊 | 彩色粒子爆炸噴射 |

---

### 🌈 九九乘法表.html — 超級絢麗九九乘法表

用純 CSS 動畫 + JavaScript 打造的互動乘法表。

**效果特色**
- 🌠 星空背景 + 七個浮動 Emoji
- 🌈 標題動態彩虹漸層文字
- 💡 表格外框霓虹光暈動畫

**互動方式**

| 操作 | 效果 |
|------|------|
| 🖱️ 懸停格子 | 同列同欄交叉高亮 + 算式泡泡 |
| 👆 點擊格子 | 彩色粒子爆炸特效 |
| 🌈 彩虹模式 | 九列各一種霓虹色 |
| 🌸 粉嫩模式 | 柔和馬卡龍配色 |
| ⚡ 霓虹模式 | 高飽和電光色 |
| 🔆 完全平方數 | 對角線數字金色發光顯示 ★ |

---

## 🚀 使用方式

直接下載 HTML 檔案，用瀏覽器開啟即可，不需要安裝任何套件！

```bash
git clone https://github.com/aifunfun/funfun.git
cd funfun
# 用瀏覽器開啟：
start 愛心.html
start 九九乘法表.html
```

## 🛠️ 技術棧

- **HTML5 Canvas** — 愛心參數方程式繪製
- **CSS3 動畫** — keyframes、漸層、backdrop-filter
- **Vanilla JavaScript** — 粒子系統、滑鼠互動、動畫循環
